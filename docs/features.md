# 機能一覧

CicadaGalleryは無料版とプレミアム版の2つのエディションがあります。

---

## 基本機能（無料版）

無料版でも多くの機能をお使いいただけます。

### 動画管理

| 機能 | 説明 |
|------|------|
| 📁 ファイル・ディレクトリ追加 | 動画ファイルを個別またはフォルダごとに追加 |
| 🖱️ ドラッグ＆ドロップ | 動画やフォルダをドラッグ＆ドロップで追加 |
| 🏷️ タグ付け | 動画にタグを付けて整理 |
| 📂 フォルダ分け | 動画を仮想フォルダで分類・管理 |
| ⭐ お気に入り機能 | お気に入りの動画をマーク |
| 🔍 検索機能 | タイトルやタグで動画を検索 |
| 🔢 ソート機能 | 追加日、ファイル名、再生時間でソート |
| 👁️ フォルダ監視 | 登録フォルダの変更を自動検知 |

### 表示・カスタマイズ

| 機能 | 説明 |
|------|------|
| 🎨 ダークモード | 目に優しいダークテーマ |
| 🖼️ サムネイルサイズ調整 | スライダーでサムネイルサイズを調整 |
| 📐 グリッド/リスト表示 | 2種類の表示モードを切り替え |
| 🌍 多言語対応 | 日本語、英語、中国語に対応 |

### データ管理

| 機能 | 説明 |
|------|------|
| 💾 SQLiteデータベース | 動画情報を軽量なSQLiteで管理 |
| 🖼️ 自動サムネイル生成 | FFmpegによる高品質サムネイル |
| 💿 自動バックアップ | 3日ごとに自動でデータベースをバックアップ |
| 🔄 手動バックアップ・復元 | いつでもバックアップを作成・復元可能 |
| 🗂️ 3世代バックアップ保持 | 最新3件のバックアップを自動管理 |
| 📏 ウィンドウサイズ記憶 | 終了時のウィンドウサイズ・位置を保存 |

### 再生機能

| 機能 | 説明 |
|------|------|
| 📺 MPV統合 | 高性能なMPVプレイヤーで動画再生 |
| 🔊 音量設定 | デフォルト音量を設定可能 |
| 🔝 最前面表示 | プレイヤーを常に最前面に表示するオプション |

### 無料版の制限

- 動画登録数: 最大100本まで
- シーン検出: 最大5シーンまで
- 評価: お気に入り（★）のみ

---

## プレミアム機能

プレミアムライセンスを購入すると、すべての機能が解放されます。

| 機能 | 説明 |
|------|------|
| 🎬 シーン検出 | 動画のシーン変化を自動検出してサムネイル生成 |
| ⭐ 1-5星評価 | 5段階の評価システム |
| 🎨 GLSLシェーダー | Anime4Kなどのカスタムシェーダー対応 |
| 🖥️ GPU高品質レンダリング | MPVのgpu-hqプロファイル |
| ♾️ 無制限の動画登録 | 100本以上の動画を登録可能 |
| ✅ 複数選択フィルター | フォルダ/タグの複数選択でフィルタリング |
| 🔀 AND/ORモード | タグのAND/OR条件切り替え |

---

## オプション設定

オプションは4つのタブに整理されています：

| タブ | 内容 |
|------|------|
| 📺 表示 | サムネイルサイズ、ダークモード、言語設定 |
| 🎬 プレイヤー | 音量、最前面表示、シェーダー設定 |
| 💾 データ | バックアップ作成・復元、前回のバックアップ日時表示 |
| 🔑 ライセンス | ライセンスキー入力、ライセンス状態確認 |
