## システム要件

### 最小要件

- **OS**: Windows 10 以降 (64-bit)
- **RAM**: 4GB
- **ストレージ**: 100MB の空き容量

### 推奨要件

- **OS**: Windows 11 (64-bit)
- **RAM**: 8GB 以上
- **ストレージ**: 500MB の空き容量
- **GPU**: DirectX 11 対応グラフィックスカード

---

## インストール方法

1. 上記のダウンロードリンクからZIPファイルをダウンロード
2. 任意のフォルダに展開
3. `CicadaGallery-*-Setup.exe` を実行
4. 手順に従ってインストールしてください

---

## 同梱コンポーネント

以下のコンポーネントが同梱されています：

- **MPV Player**: 動画再生エンジン
- **FFmpeg**: サムネイル生成・動画解析
- **GLSL-Shaders**: 高品質アップスケーリング用シェーダー（プレミアム機能）

## プレミアムライセンス

### 価格

- **個人ライセンス**：$15（買い切り）
- **商用ライセンス**：お問い合わせください

### 購入方法

**プレミアムライセンスは Gumroad を通じて販売しています。**

1. [Gumroadの商品ページ](https://tetdarth.gumroad.com/l/jmjty)からプレミアムライセンスを購入します  
2. 購入完了後、Gumroad から届くメールに記載されている **Order ID** をコピーします  
3. [ライセンス発行ページ](https://tetdarth.github.io/CicadaGallery/license)で  
   Order ID と購入時のメールアドレスを入力します  
4. **cicadagallery.app** からライセンスキーがメールで送信されます  

### ライセンスの有効化

1. アプリケーションを起動します  
2. **Options → Premium License** を開きます  
3. ライセンスキーを入力し、「Activate」をクリックします
